# fwi-water-quality

Analysis of Fire Weather Index (FWI) and water quality data.

## Project Structure

```
├── data/
│   └── raw/              ← source data (e.g. OE_Data_Periods_14.xlsx)
├── notebooks/            ← Python scripts / analysis steps
├── outputs/
│   ├── figures/          ← plots
│   └── tables/           ← summary tables
├── .gitignore
└── README.md
```
